# TOKO KELONTONG ZIZUEL

Website modern untuk Toko Kelontong Zizuel dengan sistem manajemen produk, admin panel, dan integrasi Supabase.

![Preview Website](https://via.placeholder.com/800x400/3b82f6/ffffff?text=TOKO+KELONTONG+ZIZUEL)

## ✨ Fitur Utama

- 🔐 **Sistem Login & Register** - Wajib login sebelum melihat produk
- 👑 **Panel Admin** - Akses via klik logo (khusus admin)
- 📦 **Manajemen Produk** - Tambah, edit, hapus produk
- 🖼️ **Upload Gambar** - Untuk logo dan produk
- 🎵 **Musik Latar** - Auto play untuk semua pengguna
- 💬 **Tombol WhatsApp** - Order otomatis dengan format pesan
- 💾 **Database Cloud** - Menggunakan Supabase PostgreSQL
- 📱 **Responsive Design** - Cocok di semua ukuran layar

## 🛠️ Teknologi

- HTML5
- CSS3 (Flexbox & Grid)
- JavaScript (Vanilla)
- Supabase (Database & Auth)
- Font Awesome Icons
- Google Fonts (Inter)

## 📋 Prasyarat

- Akun [Supabase](https://supabase.com)
- Akun [GitHub](https://github.com)
- Domain (opsional, bisa pake GitHub Pages)
